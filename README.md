# FivePD-OfficerDown
Officer Down / FivePD Callout

This callout will spawn a dead officer/agent inside of an FIB vehicle.

# How to install
Download this callout

Place the OfficerDown1.net.dll file in your FivePD Callouts folder

Restart FivePD
